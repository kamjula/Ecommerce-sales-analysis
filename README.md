# 🛒 E-Commerce Sales Analysis

An end-to-end sales analytics project on a multi-category e-commerce dataset — covering data cleaning, EDA, revenue trends, regional breakdowns, and business recommendations using Python.

## 🔍 What This Covers

| Analysis | What It Shows |
|----------|--------------|
| 📦 Top Products by Revenue | Ranked bar chart — which products drive the most sales |
| 🗺️ Regional Breakdown | Pie + bar charts — revenue share by U.S. region |
| 📈 Sales Trend | Daily trend line with 7-day rolling average |
| 🗂️ Category Performance | Revenue and units sold per category |
| 🔵 Revenue vs Quantity | Scatter plot by product, colored by category |
| 💡 Auto Recommendations | Business insights generated at the end of the notebook |

## 📊 Dataset

| Column | Type | Description |
|--------|------|-------------|
| OrderID | Integer | Unique order ID |
| Product | String | Shirt, Phone, Shoes, Laptop, Headphones |
| Category | String | Apparel, Electronics, Footwear |
| Sales | Float | Revenue from the order ($) |
| Quantity | Integer | Units ordered |
| OrderDate | Date | Date of order |
| Region | String | New York, Texas, California, Florida |

## 🛠 Tech Stack

Python · Pandas · NumPy · Matplotlib · Seaborn
